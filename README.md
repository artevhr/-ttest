## 🚀 Первый запуск (локально)

```bash
cd rt_bot
pip install -r requirements.txt
cp .env.example .env
# Отредактируй .env (токен, канал, ID админа)
python main.py
```

## ☁️ Деплой на Railway

1. Создай аккаунт на [railway.app](https://railway.app)
2. Нажми **New Project → Deploy from GitHub repo**
3. Подключи репозиторий с папкой `rt_bot`
4. Перейди в **Variables** и добавь:

| Переменная | Пример значения |
|---|---|
| `BOT_TOKEN` | `1234567890:ABCdef...` |
| `CHANNEL_USERNAME` | `@your_channel` |
| `ADMIN_IDS` | `123456789` |
| `DB_PATH` | `bot.db` |

5. Railway сам запустит бота через `railway.toml`

> ⚠️ Railway — бесплатный план имеет лимит часов в месяц.
> Для постоянной работы используй **Hobby plan ($5/мес)**.

## ➕ Как добавлять контент

### 📚 Добавить сборник

1. Положи PDF/Word/JPG файл в папку:
   ```
   content/collections/math/твой_файл.pdf
   ```

2. Открой `content/collections/math/index.json` и добавь запись:
   ```json
   {
     "id": "math_col_003",
     "title": "Название сборника",
     "filename": "твой_файл.pdf"
   }
   ```

**Правило именования ID:** `предмет_col_номер`
- Математика: `math_col_001`
- Русский: `russian_col_001`
- Английский: `english_col_001`

---

### ✏️ Добавить тест

Создай файл по шаблону имени:
```
content/tests/drt/math_drt_2025-2026_I_1.json
                   ↑     ↑     ↑        ↑  ↑
                предмет  тип   год     этап вариант
```

**Типы тестов:** `drt`, `rt`, `ct`, `additional`
**Предметы:** `math`, `russian`, `english`
**Этапы:** `I`, `II`, `III`

**Формат файла теста:**
```json
{
  "title": "ДРТ по математике 2025-2026, I этап, вариант 1",
  "questions": [
    {
      "id": 1,
      "text": "Текст вопроса",
      "type": "choice",
      "options": {"A": "вариант 1", "B": "вариант 2", "C": "вариант 3", "D": "вариант 4"},
      "correct": "B",
      "hint": "Объяснение как решать (из сборника)",
      "primary_points": 1
    },
    {
      "id": 2,
      "text": "Текст вопроса с вводом ответа",
      "type": "input",
      "correct": ["правильный ответ", "альтернативная запись"],
      "hint": "Подсказка",
      "primary_points": 2
    }
  ],
  "scoring_table": {
    "0": 0,
    "1": 15,
    "2": 25,
    "3": 35
  }
}
```

**`type`:**
- `"choice"` — выбор из A/B/C/D (кнопки)
- `"input"` — пользователь вводит текстом

**`correct` для input** — можно массив допустимых записей:
```json
"correct": ["2 и -2", "±2", "x=2, x=-2", "-2 и 2"]
```

**`scoring_table`** — ключ = первичный балл, значение = тестовый балл:
```json
"scoring_table": {"0":0, "1":10, "2":20, "3":30, ...}
```

### 📖 Добавить тему в сокращённый сборник

1. Открой `content/short_collections/math/index.json`, добавь:
   ```json
   {"id": "percent", "title": "📊 Проценты"}
   ```

2. Создай файл `content/short_collections/math/percent.json`:
   ```json
   {
     "title": "Проценты",
     "pages": [
       {
         "page": 1,
         "title": "Что такое процент",
         "content": "1% = 1/100 от числа.\n\n<b>Пример:</b> 20% от 50 = 50 · 0.2 = 10"
       },
       {
         "page": 2,
         "title": "Формулы",
         "content": "<b>Найти % от числа:</b>\nA · P/100\n\n<b>Найти число по %:</b>\nA / P · 100"
       }
     ]
   }
   ```

В тексте можно использовать HTML-теги: `<b>жирный</b>`, `<i>курсив</i>`, `<code>код</code>`

---

### 📌 Добавить шпаргалку

1. Положи файл (JPG, PNG, PDF) в:
   ```
   content/cheatsheets/math/formulas.jpg
   ```

2. Добавь в `content/cheatsheets/math/index.json`:
   ```json
   {
     "id": "math_cheat_003",
     "title": "Тригонометрические формулы",
     "filename": "formulas.jpg"
   }
   ```

JPG/PNG отправляются как фото, PDF — как документ.

### 🔗 Добавить полезную ссылку

Открой `handlers/links.py` и добавь в список `LINKS`:
```python
LINKS = [
    {"title": "Сайт ЦТ Беларуси",    "url": "https://ctest.by"},
    {"title": "Решу ЦТ — математика", "url": "https://math-ct.sdamgia.ru"},
    # Добавляй сюда ↓
    {"title": "Твоя ссылка",          "url": "https://example.com"},
]
```

## 👤 Как узнать свой Telegram ID

Напиши боту [@userinfobot](https://t.me/userinfobot) — он пришлёт твой ID.
Вставь его в переменную `ADMIN_IDS` в `.env`.

## 🛠 Админ-панель

Напиши в боте `/admin` (только для пользователей из `ADMIN_IDS`).

**Доступно:**
- 📊 **Статистика** — всего пользователей, активных сегодня, тестов пройдено, количество контента
- 📣 **Управление рекламой** — включить/выключить, изменить

**Настройка рекламы** — отправь 5 строк:
```
https://ссылка-на-фото.jpg   ← или - если без фото
https://t.me/канал_рекламодателя
Название канала
Текст рекламы
5
```

Последняя цифра — секунды обратного отсчёта до кнопки «Пропустить».

## 📤 Как скинуть тебе тест для конвертации

Когда скидываешь фото/PDF теста, напиши мне:
- Предмет (математика/русский/английский)
- Тип (ДРТ/РТ/ЦТ)
- Год и этап (например: 2025-2026, I этап)
- Вариант (1 или 2)
- Таблицу перевода первичного балла в тестовый (если есть)

Я верну готовый JSON-файл, который ты просто кладёшь в нужную папку.

## ⚙️ Если что-то не работает

| Проблема | Решение |
|---|---|
| Бот не отвечает | Проверь `BOT_TOKEN` в `.env` |
| «Файл не найден» | Убедись что filename в index.json совпадает с реальным именем файла |
| Тест не появляется | Проверь имя файла: `предмет_тип_год_этап_вариант.json` |
| Реклама не показывается | В админке: убедись что реклама включена |
| Подписка не проверяется | Убедись что бот — администратор в канале |
